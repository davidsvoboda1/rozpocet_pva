# Správce Osobních Financí

Jednoduchá a přehledná aplikace pro správu osobního rozpočtu. Pomůže vám sledovat vaše příjmy, výdaje a udržet si přehled o tom, za co nejvíce utrácíte.

## Hlavní funkce
* **Evidence příjmů a výdajů:** Rychlé zadávání finančních toků.
* **Kategorie:** Třídění výdajů (např. jídlo, zábava, bydlení) pro lepší přehled.
* **Aktuální bilance:** Okamžitý pohled na to, kolik peněz vám zbývá.

---

## Jak projekt stáhnout a spustit

Chcete-li si aplikaci vyzkoušet nebo upravit kód, postupujte podle následujících kroků:

### 1. Stažení projektu z GitHubu
1. Klikněte na zelené tlačítko **`Code`** v pravém horním rohu této stránky.
2. Z nabídky vyberte možnost **`Download ZIP`**.
3. Stažený soubor `.zip` uložte do svého počítače.

### 2. Rozbalení a otevření v IDE
1. Najděte stažený `.zip` soubor a **rozbalte ho** (extrahujte) do libovolné složky.
2. Otevřete program **Visual Studio**.
3. Zvolte možnost "Otevřít projekt nebo řešení" (Open a project or solution).
4. Najděte rozbalenou složku a otevřete soubor s příponou **`.csproj`** (tím se vám ve Visual Studiu načte celý projekt).

### 3. Spuštění aplikace
* Jakmile se projekt načte, stačí v horní liště Visual Studia kliknout na zelené tlačítko **`Spustit`** (nebo stisknout klávesu **F5**).
